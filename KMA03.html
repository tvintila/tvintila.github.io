TEDS2016 Multiple Regression

Use the TEDS2016 dataset to run a multiple regression model. Access the data set using the following codes. Select only relevant variables to create a subset of the dataset (Tondu, female, DPP, age, income, edu, Taiwaneseand Econ_worse). Make sure the dependent variable Tonduis coded with right labels.

Accessing and Preparing Data

library(haven)
TEDS_2016 <- read_stata("https://github.com/datageneration/home/blob/master/DataProgramming/data/TEDS_2016.dta?raw=true")

TEDS_2016$Tondu <- as.numeric(TEDS_2016$Tondu,labels=c("Unificationnow","Statusquo,unif.infuture","Statusquo,decidelater","Statusquoforever","Statusquo,indep.infuture","Independencenow","Noresponse")) 

subTEDS_2016 <- subset(TEDS_2016, select = c(Tondu, female, DPP, age, income, edu, Taiwanese, Econ_worse))

Plotting

Run a regplot on the dependent variable using:

a. Age

b. Education

c. Income

regplot=function(x,y){
  fit=lm(y~x)
  plot(x,y)
  abline(fit,col="red")
}
attach(subTEDS_2016)
regplot(age, Tondu)

regplot(edu, Tondu)

regplot(income, Tondu)
